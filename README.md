# 🎮 GameBlocker

**GameBlocker** is a lightweight Windows utility designed to combat procrastination by detecting and terminating game processes. Built to learn **C# .NET 10** and **System Architecture**.

## 🚀 Status: MVP (In Progress)
Currently a CLI tool that can:
- [x] List running user applications.
- [ ] Terminate processes by name.
- [ ] Monitor in background (Daemon mode).
- [ ] Block processes automatically based on a blocklist.

## 🛠️ Tech Stack
- **Language:** C# (.NET 10)
- **Architecture:** Console App (eventually Windows Service + React UI)
- **Native AOT:** Compiled to standalone Windows executable.

## 📦 How to Run
1. Open `GameBlocker.slnx` in Visual Studio 2022.
2. Press **F5** to build and debug.
